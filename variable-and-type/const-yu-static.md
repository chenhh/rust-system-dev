# const與static

## 簡介

在軟體開發過程中，如果一個變數總是保持不變，我們可以宣告為常量，如果一個變數全域性唯一，可以使用靜態變數。Rust語言中使用const, static來實現這2個場景，但與其他語言稍有不同，rust中的`const` 和`static`不能一起使用。

const 在 Rust 中是一個關鍵字，而且總是圍繞著常量表達式 (constant expressions) 和編譯期求值等話題。
